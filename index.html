SOAC parametrizations for accurate density functional tight binding electronic structure models

SOAC stands for Shell-Optimized Atomic Confinement, which is a procedure we applied to generating Slater-Koster parametrizations with accurate electronic structure predictions for solid compounds formed by elements across the periodic table.

Our optimization procedure is described in this article:
Balzaretti & Voss, J. Chem. Theory Comput. 2024, 20, 16, 7272â€“7286. DOI:10.1021/acs.jctc.4c00345
With this publication, we released SOAC parametrizations for all transition metals, and these parametrizations can be obtained from our SK_repository.
The parameters can be used with tight-binding codes compatible with the SKF file format, such as DFTB+ code.
